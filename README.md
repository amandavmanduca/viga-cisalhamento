
## Descrição

Desenvolve uma simples calculadora de Vigas de Concreto armado ao esforço de cisalhamento.


### Como rodar o projeto localmente
- `yarn install`
- `yarn dev`



### Tecnologias

- ReactJS [NextJS]
- CSS Modules
- TypeScript
